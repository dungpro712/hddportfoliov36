# Ho Dai Dung - Portfolio Website

A modern, interactive portfolio website built with React and Tailwind CSS. Features a fully editable interface with persistent data storage using localStorage.

## 🚀 Features

- **Interactive UI**: Smooth animations, hover effects, and transitions
- **Editable Content**: Toggle edit mode to modify all portfolio content
- **Persistent Storage**: Auto-save changes to localStorage
- **Responsive Design**: Mobile-friendly layout
- **Modern Stack**: React + Tailwind CSS + Lucide Icons
- **Project Showcase**: Display projects with details and image gallery
- **Self-Evaluation**: Strengths, weaknesses, and improvement areas
- **Development Plan**: SMART objectives and strategic roadmap
- **Timeline**: Co-operation progress visualization

## 📋 Sections

1. **Intro** - Name, title, bio, and contact information
2. **Education** - Educational background and achievements
3. **Projects** - Key projects with details and photo gallery
4. **Self-Evaluation** - Personal SWOT analysis and PDP
5. **Skills** - Hard and soft skills showcase
6. **Roadmap** - Career goals and milestones

## 🛠️ Installation

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

## 📦 Dependencies

- **React** 18.x - UI library
- **react-dom** 18.x - React DOM rendering
- **react-scripts** - Create React App scripts
- **lucide-react** - Icon library
- **tailwindcss** - Utility-first CSS framework
- **autoprefixer** - PostCSS plugin for vendor prefixes

## 🎨 Customization

Edit the `initialData` object in `src/App.jsx` to customize:
- Personal information
- Education history
- Projects and achievements
- Skills and competencies
- Career roadmap

## 📝 Usage

1. Open the portfolio in your browser
2. Click **"Edit Portfolio"** button (bottom-right)
3. Modify any content directly in the interface
4. Click **"Save Changes"** to persist your edits
5. Click **"Reset All"** to revert to defaults

## 🌐 Deployment

Deploy to:
- **Vercel**: `vercel deploy`
- **Netlify**: Drag & drop the `build` folder
- **GitHub Pages**: Use `gh-pages` package
